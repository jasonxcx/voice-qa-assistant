问题清单：
- 大模型生成回答需要将问题的回答流式地显示在字幕中，即大模型每吐一个字，字幕就跳一个字；现在会卡在”正在生成回答...“，一直不出结果；即使api已经返回了完整的结果，也不会更新字幕
- 开始监听后，只有当点击停止监听后，才将一整段语音拿去转录

AI解决：
- 主窗口模型模式为LM Studio时会刷新可选模型列表，但是如果是用户之前配置了一个模型，列表刷新后应该先去匹配是否存在，而不是默认选择第一个；即使匹配不上也应该优先用loaded_instances不为空的模型；loaded_instances不为空的模型在下拉选择时应该排在前面，且颜色做区分

人工验证：
1. 是否进行转录不再由系统根据语音大小、时间间隔自动判断 
2. 字幕窗口增加按钮，可以“开始监听”和“停止监听“ 
3. 字幕窗口样式调整：所有按钮都并排放置，翻页按钮放在右下角，”隐藏“按钮颜色调成灰色
4. 主窗口大模型设置希望是使用Ollama或者OpenAI时手动填写模型名称，使用LM Studio时通过接口查询到有哪些模型并可以下拉选择
5. 主窗口模型模式为LM Studio时，不应该显示“模型名称”
